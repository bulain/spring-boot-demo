## ADDED Requirements

### Requirement: 获取用户角色和权限
系统必须提供端点来获取分配给用户的所有角色和权限。

#### Scenario: 查询用户的完整权限集
- **WHEN** 管理员请求特定用户的权限
- **THEN** 系统返回分配给该用户的所有角色
- **AND** 系统返回从这些角色派生的所有权限（已去重）
- **AND** 系统按资源类型分组返回权限

### Requirement: 获取用户权限编码列表
系统必须提供权限编码列表用于快速查询。

#### Scenario: 获取用户权限编码
- **WHEN** 系统在访问检查期间请求用户的权限编码
- **THEN** 系统返回通过角色分配给该用户的所有权限编码的扁平化列表

### Requirement: 权限树结构
系统必须支持分层权限表示。

#### Scenario: 获取用户的权限树
- **WHEN** 管理员请求用户的权限树
- **THEN** 系统返回显示角色及其关联权限的分层结构
- **AND** 系统指示哪些角色已启用/禁用
