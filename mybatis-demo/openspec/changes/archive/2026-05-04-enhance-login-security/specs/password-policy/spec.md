## ADDED Requirements

### Requirement: Password Complexity Validation

系统 SHALL 在创建用户、修改密码、重置密码时验证密码复杂度。

#### Scenario: 密码长度不足
- **WHEN** 用户设置的密码长度小于 8 位
- **THEN** 返回错误提示："密码长度不能小于 8 位"
- **AND** 拒绝创建/修改

#### Scenario: 密码复杂度不足
- **WHEN** 用户设置的密码未包含大小写字母、数字、特殊字符中的至少 3 种
- **THEN** 返回错误提示："密码需要包含大小写字母、数字、特殊字符中的至少 3 种"
- **AND** 拒绝创建/修改

#### Scenario: 密码包含用户名
- **WHEN** 用户设置的密码中包含用户名
- **THEN** 返回错误提示："密码不能包含用户名"
- **AND** 拒绝创建/修改

#### Scenario: 密码为常见弱密码
- **WHEN** 用户设置的密码在常见弱密码列表中（如 123456、password）
- **THEN** 返回错误提示："密码过于简单，请设置更复杂的密码"
- **AND** 拒绝创建/修改

#### Scenario: 密码符合所有要求
- **WHEN** 用户设置的密码符合所有复杂度要求
- **THEN** 允许创建/修改
