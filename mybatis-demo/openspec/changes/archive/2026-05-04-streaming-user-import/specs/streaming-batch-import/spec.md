## ADDED Requirements

### Requirement: 基于回调的流式批量导入框架

系统 SHALL 提供基于 Function 回调的流式批量导入能力，可复用于其他实体。

#### Scenario: 监听器触发批次回调
- **WHEN** 监听器收集达到批次阈值（100行）
- **THEN** 立即调用回调函数处理该批次
- **AND** 清空当前批次，释放内存
- **AND** 继续处理后续行

#### Scenario: 最终批次处理
- **WHEN** 文件读取完成
- **THEN** 处理剩余的不足 100 行（部分批次）

#### Scenario: 每批次独立事务
- **WHEN** 处理单个批次
- **THEN** 该批次在独立事务中执行
- **AND** 当前批次失败不回滚已提交批次

#### Scenario: 行级数据校验
- **WHEN** 监听器读取每一行数据
- **THEN** 立即进行必填项和格式校验
- **AND** 校验失败不进入批次处理，记录错误
