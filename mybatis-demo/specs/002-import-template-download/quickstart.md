# 导入模板下载 - 快速开始

**创建日期**: 2026-05-05
**功能**: 导入模板下载

## 功能说明

本功能为系统提供用户和角色的导入模板下载功能，管理员可以下载标准的Excel模板，按照格式填写后批量导入用户和角色数据。

### 主要特性

1. **双Sheet结构**：每个模板包含"填写说明"和"数据"两个Sheet
2. **智能示例**：提供真实可用的示例数据，帮助管理员理解填写格式
3. **中文支持**：文件名、Sheet名称、字段说明均使用中文
4. **权限控制**：仅管理员可下载模板，保障系统安全

## 使用指南

### 下载用户导入模板

1. 以管理员身份登录系统
2. 进入用户管理页面
3. 点击"下载导入模板"按钮
4. 浏览器自动下载 `用户导入模板.xlsx`

**模板使用**:
1. 打开Excel文件，先阅读"填写说明"Sheet
2. 切换到"用户数据"Sheet，参考示例数据填写
3. 必填字段：用户名、姓名、密码、状态
4. 保存文件后，在系统中执行导入操作

### 下载角色导入模板

1. 以管理员身份登录系统
2. 进入角色管理页面
3. 点击"下载导入模板"按钮
4. 浏览器自动下载 `角色导入模板.xlsx`

**模板使用**:
1. 打开Excel文件，先阅读"填写说明"Sheet
2. 切换到"角色数据"Sheet，参考示例数据填写
3. 必填字段：角色编码、角色名称、状态
4. 保存文件后，在系统中执行导入操作

## 常见问题

### Q: 下载的文件无法打开？
A: 请检查下载的文件大小是否正常（通常10KB左右），如文件损坏请重新下载。建议使用Excel 2016+或WPS打开。

### Q: 导入时提示格式错误？
A: 请仔细阅读模板中的"填写说明"Sheet，确保每个字段的格式符合要求。特别注意：
- 用户名和角色编码不能重复
- 状态字段只能填写"启用"或"禁用"
- 手机号必须是11位数字

### Q: 可以修改模板的列顺序吗？
A: 不可以，导入功能按列名识别字段，修改列顺序或列名会导致导入失败。
