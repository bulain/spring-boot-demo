# 导入模板下载 - 数据模型

**创建日期**: 2026-05-05
**功能**: 导入模板下载

## 数据模型

本功能不涉及数据库表结构变更，仅涉及Excel模板的数据结构定义。

### 用户导入模板

**Sheet 1: 填写说明**

| 列名 | 说明 |
|------|------|
| 字段名称 | 列出所有字段名 |
| 是否必填 | 标记必填/可选 |
| 字段说明 | 详细解释每个字段的格式要求和约束 |
| 示例值 | 提供正确的填写示例 |

**Sheet 2: 用户数据**

| 列名 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 用户名 | String | 是 | 系统唯一标识，字母数字组合 | admin |
| 姓名 | String | 是 | 用户真实姓名 | 张三 |
| 邮箱 | String | 否 | 有效邮箱格式 | zhangsan@example.com |
| 手机号 | String | 否 | 11位手机号 | 13800138000 |
| 密码 | String | 是 | 明文填写，导入时系统自动加密 | 123456 |
| 状态 | String | 是 | 启用/禁用 | 启用 |

### 角色导入模板

**Sheet 1: 填写说明**

| 列名 | 说明 |
|------|------|
| 字段名称 | 列出所有字段名 |
| 是否必填 | 标记必填/可选 |
| 字段说明 | 详细解释每个字段的格式要求和约束 |
| 示例值 | 提供正确的填写示例 |

**Sheet 2: 角色数据**

| 列名 | 类型 | 必填 | 说明 | 示例 |
|------|------|------|------|------|
| 角色编码 | String | 是 | 系统唯一标识，字母下划线 | ROLE_ADMIN |
| 角色名称 | String | 是 | 角色显示名称 | 系统管理员 |
| 描述 | String | 否 | 角色功能描述 | 管理系统所有功能 |
| 状态 | String | 是 | 启用/禁用 | 启用 |

## 验证规则

1. **字符串长度限制**
   - 用户名：3-50字符
   - 姓名：2-50字符
   - 角色编码：3-100字符
   - 角色名称：2-50字符

2. **格式验证**
   - 邮箱：符合RFC 5322标准
   - 手机号：中国大陆11位手机号格式
   - 状态字段：只能是"启用"或"禁用"

3. **唯一性约束**
   - 用户名在系统中唯一
   - 角色编码在系统中唯一

## 文件格式要求

- 文件格式：Excel 2007+ (.xlsx)
- 编码：UTF-8
- 文件大小：无特殊限制（通常<100KB）
- Sheet名称：固定为"填写说明"和"数据"
