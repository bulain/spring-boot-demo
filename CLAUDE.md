# CLAUDE.md

本文档为 Claude Code 提供本仓库的操作指南和项目说明。

## 项目概述

- 多模块 Maven Spring Boot 示例项目（Spring Boot 3.5.14，Java 17）
- 各模块在根 pom.xml 中配置，目前仅 mybatis-demo 激活
- 其他模块（admin-demo、cache-demo、shiro-demo、flowable-demo 等）均已注释

## 模块概览

| 模块 | 状态 | 说明 |
|------|------|------|
| mybatis-demo | 激活 | MyBatis Plus 多数据库演示 |
| 其他*-demo | 禁用 | 各技术栈演示，仅供参考 |
